
**Бинарное деерво поиска** - дерево, построенное по правилу: узел с меньшим значением, чем у корня вставляется слева, а с большим справа.

```python
from bst import BST


bst = BST()
bst.insert_list([5, 2, 1, 3, 6, 4])
bst.visualize()
```
